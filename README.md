# BookFlix

A simple, responsive landing page for a fictional bookstore built with Next.js and Tailwind CSS.

## Features

- **Hero Section** — Bookstore name, tagline, and a CTA button to browse books
- **Featured Books** — 6 book cards fetched from the [Open Library API](https://openlibrary.org/), each displaying cover image, title, author, price, and a "Buy Now" button
- **About Page** — Short description of the bookstore
- **Footer** — Social links, contact email, and copyright
- **Dark Mode Toggle** — Switch between light and dark themes, persisted in localStorage
- **Responsive Design** — Mobile-friendly with a collapsible hamburger menu
- **Hover Effects** — Scale animations on book cards, color transitions on links and buttons

## Tech Stack

- [Next.js](https://nextjs.org/) 16
- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/) 4
- [Base UI](https://base-ui.com/) (button & switch components)
- [Lucide React](https://lucide.dev/) (icons)
- [Open Library API](https://openlibrary.org/developers/api) (book data & covers)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
